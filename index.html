Alle offiziellen Befehle und Features auf einen Blick erklärt.

/clear

Chat-Reinigung

Hält deinen Server sauber. Mit diesem Befehl kannst du eine spezifische Anzahl an Nachrichten blitzschnell löschen. Ideal, um Spam zu entfernen oder Kanäle für neue Ankündigungen vorzubereiten.

/dice

Glückswürfel

Bringt Action in den Chat! Der Bot generiert eine Zufallszahl. Perfekt für kleine Gewinnspiele, Entscheidungen innerhalb der Community oder einfach zum Zeitvertreib zwischendurch.

/serverinfo

Server-Details

Zeigt alle wichtigen Daten deines Discord-Servers in einer eleganten Übersicht: Erstellungsdatum, Mitgliederanzahl, Boost-Status und die Anzahl der Kanäle auf einen Klick.

/ticket

Support-System

Das Herzstück für Admins. Erstellt ein interaktives Ticket-Panel. User können per Knopfdruck private Support-Räume öffnen, in denen dein Team direkt helfen kann – inklusive Archivierung.

/userinfo

Nutzer-Profil

Erhalte sofort Informationen über ein Mitglied. Sieh dir an, wann der User Discord beigetreten ist, welche Rollen er besitzt und erhalte die eindeutige User-ID für administrative Zwecke.
